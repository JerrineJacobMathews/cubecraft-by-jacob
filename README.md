# CubeCraft by Jacob

CubeCraft by Jacob is a modern Rubik’s Cube solver web app designed for GitHub Pages.

## Current status
This version includes:
- branded landing page
- real interactive 3D cube viewer
- random scramble generation
- real solver initialization
- computed solution playback
- reset to solved state
- polished dark UI

## Current solve mode
This version solves generated 3×3 scrambles inside the browser.
