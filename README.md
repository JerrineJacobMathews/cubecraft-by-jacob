# CubeCraft by Jacob

CubeCraft by Jacob is a modern Rubik’s Cube solver web app designed for GitHub Pages.

## Current status
This version includes:
- branded landing page
- real interactive 3D cube viewer
- random scramble generation
- manual sticker input
- live color counts
- deeper validation
- solve generated scrambles
- solve manually entered cube states
- polished dark UI

## Current solve modes
1. Generated scramble solve
2. Manual cube-state solve

## Validation improvements
- checks missing stickers
- checks per-color counts
- checks locked centers
- checks solver parsing
- checks apparent solvability

## UX improvements
- selected paint label
- empty paint mode
- right-click to clear
- live counts
- validation banner
